# Projet sklearn – Prédiction du Nasdaq

Ce projet a pour objectif de mettre en place un modèle de machine learning
afin de prédire le rendement journalier de l’indice Nasdaq à partir de données historiques.

## Technologies utilisées
- Python
- pandas
- scikit-learn
- yfinance

## Structure du projet
- data/ : données
- src/ : code Python
- notebooks/ : exploration et tests
